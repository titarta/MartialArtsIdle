# Alchemy

Pill crafting system accessed via the **Production > Alchemy** tab. Combine three herbs in the Refining Furnace to produce cultivation pills with temporary buffs.

---

## How It Works

1. Select **three herbs** (slots arranged in a triangle around the furnace)
2. The combination determines which pill is produced — every valid combination maps to exactly one pill
3. Click **Craft** to consume the herbs and receive the pill
4. Use pills from the **Home screen** below the cultivation area

---

## Recipe Rules

- A recipe is any **unordered combination of 3 herbs** (duplicates allowed — you can use the same herb three times)
- **Rarity constraint**: the maximum rarity distance between any two herbs in a recipe must be **≤ 2 tiers** (e.g. Iron + Silver is allowed, but Iron + Gold is not)
- There are **92 valid combinations** across the 8 available herbs
- Each combination produces **exactly one pill** — no ambiguity

### Rarity Tiers

| Tier | Rarity | Herbs |
|---|---|---|
| 1 | Iron | Soul Calming Grass |
| 2 | Bronze | Jade Heart Flower, Netherworld Flame Mushroom |
| 3 | Silver | Thousand-Year Ginseng, Blood Lotus, Dragon Saliva Grass |
| 4 | Gold | Purple Cloud Vine |
| 5 | Transcendent | Immortal Revival Leaf |

### Pill Rarity from Ingredients

The pill's rarity is determined by the **total rarity tier sum** of the three herbs:

| Tier Sum | Pill Rarity |
|---|---|
| 3–5 | Iron |
| 6–7 | Bronze |
| 8–9 | Silver |
| 10–11 | Gold |
| 12–15 | Transcendent |

## Pill Usage

- Pills appear on the **Home screen** below cultivation
- Click a pill to **consume** it — the buff activates immediately
- **Multiple pills can be active simultaneously** — their effects stack
- Active buffs display with a countdown timer
- Hover/hold a pill to see its effects before using
